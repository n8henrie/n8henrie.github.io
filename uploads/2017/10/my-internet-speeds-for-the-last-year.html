Home Assistant Data¶

hass_speeds.csv comes from Home Assistant's Fast.com sensor. "Unknown" speeds usually mean that either the internet was out, (or possibly that I was working on the modem, which was usually because the internet was out), so I am replacing these values with 0.

In [4]:
hass_df = pd.read_csv(
    "hass_speeds.csv",
    parse_dates=[1],
    index_col=1,
    header=0,
    names=["Speed", "Date"],
    converters={'Speed': lambda val: 0. if val == 'unknown' else float(val)},
)
